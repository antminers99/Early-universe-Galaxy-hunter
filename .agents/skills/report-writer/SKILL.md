# Report Writer

## Purpose

Write clean, well-structured reports that communicate findings clearly.

## When to Use

- Analysis phase is complete and results need formal documentation
- Preparing a summary for collaborators
- Writing up a finding for review
- Creating an internal progress report

## Report Structure

```
# [عنوان التقرير]

## ملخص تنفيذي
[2-3 فقرات تلخص كل شيء — يقرأها من لا يملك وقتًا للتفاصيل]

## المقدمة
[السياق والدافع — لماذا هذا مهم]

## البيانات والمنهجية
### البيانات المستخدمة
[المصادر والأحجام والصيغ]

### المنهجية
[ماذا فعلنا ولماذا اخترنا هذا النهج]

## النتائج
### النتيجة الرئيسية
[أهم ما وجدناه]

### النتائج الثانوية
[نتائج إضافية]

## المناقشة
### التفسير
[ماذا تعني النتائج]

### المحدوديات
[ما لم نستطع فعله وماذا يمكن أن يكون خاطئًا]

### المقارنة مع الأعمال السابقة
[كيف تقارن نتائجنا مع الآخرين]

## الاستنتاجات
[الرسالة الرئيسية — بجملتين]

## الخطوات التالية
[ماذا نقترح]

## المراجع
[الأوراق المستشهد بها]
```

## Language Rules

- Main text in Arabic
- Technical terms can be in English with Arabic explanation
- Figures labeled in both Arabic and English
- Claim language calibrated with Claim Limiter
- Honest about limitations
